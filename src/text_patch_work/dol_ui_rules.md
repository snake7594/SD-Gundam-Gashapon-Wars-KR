# main.dol UI 문자열 일본어→한국어 번역 규칙

게임 「SD건담 가샤폰 워즈」의 시스템/메뉴/버튼 UI 텍스트입니다. 짧고 명확하게.

## 절대 규칙
1. **바이트 예산 준수(중요)**: 각 항목의 `budget`(바이트)를 넘으면 안 됩니다.
   - 한글 1글자 = 2바이트, 영숫자/기호(반각) = 1바이트.
   - 즉 한글은 최대 `budget/2` 글자. 넘으면 게임이 깨집니다. 짧게!
2. **제어토큰 보존**: `@c0`~`@cf`, `@b0`~`@bf`, `@L`, `@i6` 등 `@`+영숫자 코드는 그대로, 같은 위치에.
3. **후리가나 제거**: `|漢字(かな)` → 뜻을 한국어로. `|`와 괄호 가나 삭제.
   예) `|選択(せんたく)へ|戻(もど)る` → `선택으로 돌아가기` (예산 맞으면) / 넘으면 `선택 복귀`
4. **줄바꿈 `\n`** 있으면 유지.
5. **전각영숫자**(ＳＹＳＴＥＭ, ２Ｄ 등)와 기호는 그대로 두거나 자연스럽게.
6. **고유명사**: ブルーサイド→블루사이드, レッドサイド→레드사이드, ＣＯＭ→COM, ガシャポリス→가샤폴리스.

## 표준 UI 용어 (짧게)
- はい→예, いいえ→아니오, もどる→뒤로, けってい→결정, キャンセル→취소
- セーブ→저장, ロード→불러오기, メモリーカード→메모리카드, よろしいですか→하시겠습니까
- ヘルプ→도움말, リトライ→재시도, ポーズメニュー→일시정지, ＳＹＳＴＥＭ→SYSTEM(그대로)
- ゲーム再開→게임 재개, ゲームの終了→게임 종료, 操作説明→조작설명, クリア条件→클리어 조건
- ユニット→유닛, カプセル→캡슐, フィールド→필드, ミッション→미션, アイテム→아이템
- 選択→선택, 戻る→돌아가기/복귀, 編集→편집, 入力→입력, 設定→설정, 表示→표시
- おまかせ→맡기기, もとにもどす→되돌리기, ならびかえ→정렬, ぞくせい→속성, ばんごう→번호
- 大気圏→대기권, コスト→코스트, プログレッシブモード→프로그레시브 모드
- 전투커맨드: まかせた→맡긴다, とどめをさせ→끝장내라, リーダーねらえ→리더 노려라,
  つっこめ→돌격, うちまくれ→쏴갈겨라, ガードせよ→가드하라, アイテムのがすな→아이템 놓치지마

## 출력
입력 각 {id, jp, budget} 에 대해 {id, ko}. **오직 JSON 배열**만 출력 파일에 쓰기. 주석/펜스 금지.
budget을 반드시 지키세요(한글 byte 계산). 애매하면 더 짧게.
