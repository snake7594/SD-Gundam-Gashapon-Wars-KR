# SPB 대사 일본어→한국어 번역 지침

당신은 GameCube 게임 「SD건담 가샤폰 워즈」의 게임 내 대사를 한국어로 번역합니다.
아동/청소년 대상의 건담 SRPG입니다. 튜토리얼(마류·키라 안내)은 친근한 존댓말(~요/~해요),
전투/스토리 대사는 캐릭터 성격에 맞게 자연스럽게 번역하세요.

## 절대 규칙 (어기면 게임이 깨집니다)

1. **제어 토큰 원형 유지**: `@c7` `@c6` `@c0` `@b4` `@b5` `@i6` 같은 `@`+2글자 코드는
   **그대로, 같은 위치에** 두세요. 절대 번역/삭제/변형 금지.
   예) `@c6「ユニット」@c7を` → `@c6「유닛」@c7을`

2. **줄바꿈 `\n` 유지**: 본문에 나오는 `\n`(역슬래시+n)은 줄바꿈입니다.
   **원문과 같은 개수·같은 위치**로 유지하세요. 한 화면(텍스트박스)에 들어가야 하므로
   줄 구조를 최대한 지키세요. (원문이 `\n` 없으면 한 줄로.)

3. **후리가나 루비 제거**: `|漢字(かな)` 형태의 루비는 **한국어로는 필요 없습니다**.
   `|` 와 `(...)` 안의 가나를 모두 없애고, 한자의 **뜻을 한국어로** 옮기세요.
   예) `|説明(せつめい)します` → `설명할게요`
   예) `|戦(たたか)いかた` → `싸우는 법`

4. **괄호/기호 유지**: `「」` `【】` `・` `…` `。` `、` `！` `？` 는 그대로 사용하세요.
   `【アムロ】` 처럼 【】 안의 화자 이름은 용어집대로 번역: `【아무로】`
   문장 끝 `。` 는 한국어에서 생략하거나 그대로 둬도 됩니다(원문 유지 권장).

5. **간결하게**: 텍스트박스가 작습니다. 뜻을 살리되 너무 늘어지지 않게.
   원문보다 글자 수가 크게 늘지 않도록 하세요.

6. **고유명사**: 반드시 glossary.md 표기를 따르세요.

## 출력 형식 (엄격)

입력 배치의 각 항목 `{id, jp}` 에 대해 `{id, ko}` 를 만들어,
**오직 JSON 배열만** 출력 파일에 쓰세요. 설명·주석·코드펜스 금지.

```
[
 {"id": 0, "ko": "@c7처음 하는 캡슐 편집이니까\\n조금만 설명할게요."},
 {"id": 20, "ko": "..."}
]
```

주의: JSON 문자열 안에서 줄바꿈 토큰은 `\\n` (JSON 이스케이프된 역슬래시+n) 으로 써야
파일에 실제로 `\n`(역슬래시+n) 두 글자로 저장됩니다. 원문 jp에 들어있던 형태를 그대로 두면 됩니다.

## 번역 예시
- jp: `@c0しきかんの\nマリュー・ラミアスです。`
  ko: `@c0함장인\n마류 라미아스입니다.`
- jp: `@c7【アムロ】「きたー！」`
  ko: `@c7【아무로】「온다ーー!」` (또는 「왔다!」)
- jp: `@c6「カーソル」@c7をこの@c6「ユニット」@c7に|合(あ)わせて\n@b4をおしてね。`
  ko: `@c6「커서」@c7를 이\n@c6「유닛」@c7에 맞추고\n@b4를 눌러줘.`
