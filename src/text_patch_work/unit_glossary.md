# 건담 유닛명 한국어 표기 규칙

카타카나 유닛명을 한국 정발/통용 표기로. 접미/기호는 아래 규칙대로 보존.

## 대표 표기 (참고)
- ガンダム→건담, ガンキャノン→건캐논, ガンタンク→건탱크, ジム→짐, ザク→자쿠
- グフ→구프, ドム→돔, リックドム→릭돔, ゲルググ→겔구그, ギャン→걍, ズゴック→즈고크
- ゾック→조크, アッガイ→앗가이, ジオング→지옹, エルメス→엘메스, ビグ・ザム→빅잠
- ケンプファー→켐퍼, アレックス→알렉스, ザクレロ→자크렐로, マドロック→매드록
- コア・ブースター→코어 부스터, ザクＦ型→자쿠 F형, シャア専用→샤아 전용
- キュベレイ→큐베레이, ハンマ・ハンマ→함마 함마, キュベレイＭｋ－Ⅱ→큐베레이 Mk-Ⅱ
- ストライク→스트라이크, エールストライク→에일 스트라이크, フリーダム→프리덤
- ジャスティス→저스티스, プロヴィデンス→프로비던스, イージス→이지스, デュエル→듀얼
- ブリッツ→블리츠, バスター→버스터, フォビドゥン→포비든, カラミティ→캘러미티
- アストレイ→아스트레이, ストライクダガー→스트라이크 대거, ダガー→대거
- ザクウォーリア/ザクウオーリア→자쿠 워리어, ザクファントム→자쿠 팬텀
- グフイグナイテッド→구프 이그나이티드, サイコガンダム→사이코 건담
- リック・ディアス→릭 디아스, リ・ガズィ→리 가지, バウンド・ドッグ→바운드 독
- ムラサメ→무라사메, ミーア→미아, パーフェクトガンダム→퍼펙트 건담
- νガンダム→ν건담, νは그대로 ν 유지

## 접미/기호 보존 규칙 (절대 유지)
- `[GOLD]`→`[GOLD]`,  `[黒]`→`[흑]`,  `[赤]`→`[적]`,  `[緑]`→`[녹]`,  `[黄]`→`[황]`,  `[白]`→`[백]`,  `[青]`→`[청]`,  `[SPARK]`→`[SPARK]`
- `／ＭＡ`→`/MA`,  `／空中`→`/공중`,  `／ＢＷＳ`→`/BWS`
- `Ｍｋ－Ⅱ`→`Mk-Ⅱ`(그대로),  `Ｅｚ８`→`Ez8`,  숫자·로마자·Ⅱ 등은 그대로
- `・`(가운뎃점)은 `・`(U+30FB) 그대로 (한글 사이 공백 대체 가능)
- 되도록 원문과 길이 비슷하게(칸에 들어가야 함).
